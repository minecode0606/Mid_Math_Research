# MIFC Project

MIFC Project(Minseo's Interesting Function Collection)의 초기 설립의도는 수학문제를 풀며 접했던
여려가지 도형이나 그림을 함수화 시켜 하나의 방정식을 유도하고 연구하는 프로젝트였습니다.
하지만 현재 MIFC 프로젝트는 단지 함수에만 국한하지 않고 다양한 수학분야에 대해 연구하고 있습니다.
